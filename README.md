## 🚀 Featured Project

### 📊 [Анализ финансовой устойчивости строительных МСП России (2021–2025)](https://github.com/xTravisBickley/construction-sme-analytics)

Исследование рентабельности 8 000+ малых и средних строительных компаний на основе открытых данных ФНС.

**Что сделано:**
- 🕷️ Спарсил финансовые отчётности через Selenium (ETL-пайплайн: сбор → очистка → трансформация → загрузка)
- 📈 Рассчитал ключевые метрики: медианная EBIT, доля убыточных компаний, квартили, отношение выплат к EBIT
- 🔍 Провёл ad-hoc исследование причин деградации метрик в 2025 году (влияние отмены льготной ипотеки и роста ключевой ставки ЦБ)
- 📊 Построил дашборды: динамика по годам, сегментация МСП, региональный срез, распределение роста EBIT
- 💡 Сформулировал рекомендации для трёх групп стейкхолдеров: банки, инвесторы, сами МСП
